Mightty React Redux Starter: gulp simple app starter for react, react-router, and redux

## Requirements
__node v4.5.0__ or higher for (partial) ES2015 support
(IE > 11, Firefox > 45, or Chrome > 55)

```

$ yarn

$ gulp 6to5

// setup dist folder
$ gulp
```
